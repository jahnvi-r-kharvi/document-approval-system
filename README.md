# 📄 Document Approval System

 Project URL
The application runs locally at:
 http://localhost:3000/

---
 Overview
The Document Approval System is a web-based application that allows users to submit documents and enables an admin to review, approve, or reject them. The system helps in maintaining a structured workflow for document management.

---
 Objectives
* Automate document submission and approval process
* Reduce manual paperwork
* Track document status efficiently

---
 Features
*  Upload documents with title
*  View all submitted documents
*  Approve documents
*  Reject documents
*  Status tracking (Pending / Approved / Rejected)
*  View uploaded files

 Technologies Used
* Frontend: HTML, CSS
* Backend: Node.js
* Framework: Express.js
* Version Control: Git & GitHub

 Workflow
1. User uploads a document
2. Document status is set to **Pending**
3. Admin reviews the document
4. Admin approves or rejects the document
5. Status is updated and displayed
